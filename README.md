家族向け食事スケジュール管理アプリケーション

家族間での「今日のご飯いる？」「何時に帰ってくる？」といった連絡や確認の手間を減らし、食事予定や帰宅時間、メッセージを円滑に共有するためのアプリケーションです。

・プログラムの操作方法（アプリの使用方法）
①アカウント登録と確認
1. 新規ユーザー登録  
   アプリ起動後、表示される「ユーザー登録」画面にて、ユーザー名を入力し「登録」ボタンを押します。
2. アカウントIDの確認  
   登録完了後、画面上部の「登録情報」ボタンを押すことで、システムから割り当てられた自身のアカウントIDを確認できます。

② 食事予定の登録・変更（個人カレンダー）
1. 日付の選択  
   カレンダー画面から、予定を登録したい日付を選択し「食事予定を登録」ボタンを押します。
2. 予定・帰宅時間・メッセージの入力 
   - 朝食・昼食・夕食それぞれについて「必要 / 不要」を選択します。
   - 「必要」を選択した場合、メッセージ欄や帰宅時間（任意）の入力欄が表示されるため、伝言や帰宅予定時間を設定して「登録する」を押します。
3. 予定の変更・更新
   日付下に表示された自分の予定カードをクリックすると、再度入力画面に遷移し、内容を更新・再保存できます。

③ 家族グループの作成とメンバー共有
1. グループ作成
   画面上部の「グループ作成」ボタンを押し、家族共通のグループ名を入力して「作成」を押します。作成と同時にグループ専用のカレンダーへ切り替わります。
2. メンバーの追加
   - 招待したい家族の端末で同様にアカウント登録を行っておきます。
   - グループ画面上部の「メンバー追加」ボタンを押し、招待したい家族のアカウントIDを入力して「追加」を押します。
3. グループでの予定共有  
   同じグループに属するメンバー全員の食事予定・帰宅時間・メッセージが、選択した日付の下に表示されます。

④ グループ脱退とアカウント削除
1. グループ退出
   「グループ退出」ボタンを押し、確認画面で「退出する」を選択すると、個人のカレンダーに戻り、グループからの共有が解除されます。
2. アカウント削除
   「登録情報」画面内の「アカウント削除」ボタンを押すと、確認ダイアログが表示されます。「OK」を選択するとアカウントおよび関連データが安全に削除され、最初のユーザー登録画面に戻ります。
